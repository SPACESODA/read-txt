# Read TXT

**Read TXT** is a lightweight text-to-speech web app that lets you listen to any text, with auto language detection, smart chunking, and offline-ready PWA support.

For the best voice quality, use Microsoft Edge or Google Chrome.

## Features

- Turn pasted text into natural speech (Edge and Chrome only).
- Drag and drop .txt or .md files.
- Auto-detect language and match a suitable voice.
- Split long content for smoother listening.
- Save multiple sections and pick up where you left off.
- Keyboard-friendly playback controls.
- Works offline after the first load (PWA ready, HTTPS or localhost required).
